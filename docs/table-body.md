# Table Body
{: .no_toc }

O `m-tbody` é usado como corpo da `m-table`.
## Propriedades

| Propriedade  | Descrição                 | Tipo     | Default     |
| ------------ | ------------------------- | -------- | ----------- |
| `body`       | Linhas do body da tabela  | `array`  | `undefined` |
| `header`     | Células do body da tabela | `array`  | `undefined` |
